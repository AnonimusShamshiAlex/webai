
# 🤖 Dr TITI AI Chat

[![Python](https://img.shields.io/badge/Python-3.8%2B-blue)](https://python.org)
[![FastAPI](https://img.shields.io/badge/FastAPI-0.115.0-009688)](https://fastapi.tiangolo.com)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![OpenRouter](https://img.shields.io/badge/OpenRouter-AI-ff6b6b)](https://openrouter.ai)

**Dr TITI** — это современный AI-чат с поддержкой текста, голоса и аудиофайлов. Работает через OpenRouter API, имеет красивый интерфейс в стиле ChatGPT и может быть опубликован в интернет через ngrok за 1 минуту.

**Разработка Dr paradox** 🧪

---

## ✨ Возможности

- 💬 **Текстовый чат** с AI (модель `arcee-ai/trinity-large-preview`)
- 🎤 **Голосовой ввод** прямо в браузере
- 📁 **Загрузка аудиофайлов** (распознавание речи через Google Speech)
- 🌐 **Публичный доступ** через ngrok (без статического IP)
- 🎨 **Красивый интерфейс** в тёмной теме (как у ChatGPT)
- 📜 **История сообщений** сохраняется в сессии
- 🔄 **Drag & drop** аудиофайлов
- ⌨️ **Горячие клавиши** (Enter для отправки, Shift+Enter для новой строки)

## 🚀 Быстрый старт

### 1. Клонирование репозитория

```bash
git clone https://github.com/твой-username/dr-titi-chat.git
cd dr-titi-chat
```

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

**Файл `requirements.txt`** (создай его в корне проекта):
```
fastapi==0.115.0
uvicorn[standard]==0.30.0
speechrecognition==3.10.0
aiohttp==3.9.5
python-multipart==0.0.9
requests==2.31.0
```

### 3. Запуск сервера

```bash
python sreverngrok.py
```

После запуска ты увидишь красивое ASCII-приветствие с адресами:


## 📦 Настройка

### 🔑 API ключ OpenRouter

В коде уже вставлен твой ключ:
```python
OPENROUTER_API_KEY = "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"
```

Если захочешь сменить модель — измени параметр `"model"` в функции `get_ai_response()`.

### 🌍 Публикация через ngrok

1. **Скачай ngrok** с [официального сайта](https://ngrok.com/download)
2. Распакуй и добавь в `PATH` или положи в папку проекта
3. При запуске сервер автоматически создаст туннель

Если ngrok не установлен — сервер запустится локально, и ты увидишь предупреждение.

## 🎮 Использование

### Веб-интерфейс

Открой браузер и перейди на любой из адресов:
- `http://localhost:5000`
- `http://[твой-ip]:5000`
- `https://[ngrok-адрес].ngrok-free.app`

### Текстовый чат
1. Введи сообщение в поле ввода
2. Нажми **Enter** (или кнопку ➤)
3. Дождись ответа AI

### Голосовой ввод
1. Нажми кнопку **🎤**
2. Разреши доступ к микрофону
3. Говори — запись начнётся автоматически
4. Нажми **🎤** ещё раз для остановки и отправки

### Загрузка аудиофайла
1. Нажми кнопку **📎** и выбери файл
2. Или просто **перетащи** аудиофайл в окно браузера
3. Нажми ➤ для отправки

### Горячие клавиши
| Действие | Клавиша |
|----------|---------|
| Отправка сообщения | `Enter` |
| Новая строка | `Shift + Enter` |
| Отмена файла | Кнопка ✕ |

## 🧠 API Endpoints

### `GET /`
Главная страница чата.

### `POST /process_text`
Обработка текстового сообщения.
```json
{
  "text": "Привет!"
}
```

### `POST /process_audio`
Обработка аудиофайла (multipart/form-data).

### `GET /history`
Получение истории сообщений.

### `GET /ngrok_url`
Получение публичного URL ngrok.

### `GET /health`
Проверка статуса сервера.

## 🛠️ Возможные проблемы и решения

| Проблема | Решение |
|----------|---------|
| **ngrok не запускается** | Установи ngrok вручную с [ngrok.com](https://ngrok.com/download) |
| **Ошибка при распознавании речи** | Проверь подключение к интернету, Google Speech требует сеть |
| **Не работает микрофон в браузере** | Разреши доступ к микрофону в настройках сайта |
| **AI отвечает на английском** | В коде указан русский язык, проверь параметры модели |
